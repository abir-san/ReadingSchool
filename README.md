#ReadingSchool

Reading School involves a pixel scrolling game similar to Champion Island involving crazy lore, quests and tasks.

Organisation

You have been invited to help in a PROJECT!

You can access the Codespace for this project by clicking code and codespace and start working in main.py using the VSC editor. 

In this project, Reading School will be recreated so that it is more creepier in a way - locked doors, marching students. A Demon Headmaster has took over the school and the real Headmaster is trapped underneath the cloisters (near Big School) in a dungeon.

There is a group of students trying to free the Headmaster and escape the school forever. Although materials are needed to escape the school like a key or a ladder, this needs to be done without getting caught!
